## Installation

Install the following packages:

**With pnpm**

```shell
pnpm add @hopper-ui/svgs
```

**With yarn**

```shell
yarn add -D @hopper-ui/svgs
```

**With npm**

```shell
npm install -D @hopper-ui/svgs
```
