# Installation

## Package manager

Run the following command to install the tokens package.

### Pnpm

```shell
pnpm add @hopper-ui/tokens
```

### Yarn

```shell
yarn add @hopper-ui/tokens
```

### With Npm

```shell
npm add @hopper-ui/tokens
```
