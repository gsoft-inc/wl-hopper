# Installation

We provide a few different ways to install tokens. This guide will walk you through the different options and help you choose the right one for your project.

## Package manager

Using your package manager of choice, run the following command to install the package.

### With pnpm

```bash
pnpm add @hopper-ui/tokens
```

### With yarn

```bash
yarn add @hopper-ui/tokens
```

### With npm

```bash
npm add @hopper-ui/tokens
```
