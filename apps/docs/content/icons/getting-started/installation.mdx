# Installation

Using your package manager of choice, run the following command to install the icons package.

## React Components

If you are using React, you can install the React components package.

<Tabs tabs={installMethods}>
    <div>
    ```shell
    pnpm add @hopper-ui/icons
    ```
</div>
    <div>
    ```shell
    yarn add -D @hopper-ui/icons
    ```
    </div>
    <div>
    ```shell
    npm add -D @hopper-ui/icons
    ```
    </div>
</Tabs>

## SVG Icons

If you are not using React or you need to use the SVG icons directly, you can install the SVG icons package.

<Tabs tabs={installMethods}>
    <div>
    ```shell
    pnpm add @hopper-ui/svg-icons
    ```
    </div>
    <div>
    ```shell
    yarn add -D @hopper-ui/svg-icons
    ```
    </div>
    <div>
    ```shell
    npm add -D @hopper-ui/svg-icons
    ```
    </div>
</Tabs>
